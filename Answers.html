В чем отличие Git от GitHub?

Git - это набор программ, которые позволяют вносить изменения в любую версию одной и той же программы во времени, отслеживать эти изменения и возвращаться в любую версию программы, независимо от того когда она была создана. GitHub - это он-лайн сервис, облегчающий работу с Git и дающий возможность одновременно нескольким разработчикам работать над одним проэктом

Что такое тег? Что такое атрибут?

Тег - это фрагмент языка гипертекстовой разметки текста. Атрибут - это свойство тега, дополняющее и расширяющее существующие функции тега.

Чем отличается язык разметки от языка програмирования? HTML можно назвать язіком программирования? Аргументируйте свою позицию?

Язык разметки предназначен для передачи информации о выводе текста, его структуре и свойствах. Язык программирования представляет собой набор последовательно выпоняемых команд, с целью обработки данных и получения расчетных результатов. HTML является прежде всего языком разметки, поэтому к языку программирования его отнести нельзя.

Зачем нужен атрибут alt у изображений?

Это атрибут для тега img. Краткое описание того, что изображено на картинке. Позволяет поисковой системе получить более полную инфорамцию о графическом изображении и оптимизировать свою работу.
